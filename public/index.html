MCP Wrapper

Convert Solidity ABI to MCP(Model Context Protocol) server

  1. Get an Invite Code: Obtain an invite code from your friend or apply by filling this form. Each code can: create up to 100 servers, invoke MCP tools for 10000 times.
  2. Smart Contract Interaction Support:
    • Read Operations: Full support for all view/pure functions (constant functions that don't modify state)
    • Write Operations: Two-step process for state-modifying functions:
      1. Transaction Construction: Generate the raw transaction data by using tool of constructTransactionData
      2. Transaction signing requires your private key and must be handled securely on your side
      3. Transaction Broadcasting: Send your signed transaction to the network by using tool of sendSignedTransaction
  3. Create a New Server: Use your invite code to create a new MCP server by providing:
    • Invite Code
    • Server Name
    • Chain RPC URL
    • Smart Contract ABI
  4. Access Your Servers: Enter your invite code to view all servers created with that code.
